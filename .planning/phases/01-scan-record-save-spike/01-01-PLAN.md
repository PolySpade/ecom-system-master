barcode_handler.dart (port of ecom-py/barcode_handler.py — ONLY 3 actions, no standalone `stop`):
```dart
enum BarcodeAction { start, stopAndStart, invalid }
class BarcodeHandler {
  String? lastBarcode;
  bool validateBarcode(String? barcode);              // non-null, non-empty after .trim()
  ({BarcodeAction action, String barcode}) processBarcode(String barcode, {required bool isRecording});
  // normalization order: (1) validate, (2) .trim().toUpperCase(), (3) sanitize path separators, (4) return action

}
```
